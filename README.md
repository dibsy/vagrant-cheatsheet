# vagrant-cheatsheet
Vagrant CheatSheet

```
vagrant box list - List of boxes added
vagrant init hashicorp/precise64

vagrant init hashicorp/precise64 
vagrant up
vagrant ssh
$exit

vagrant global-status - 
vagrant status - To check status
vagrant suspend - To sleep
vagrant halt - To shutdown
vagrant destroy - To destroy the virtual image

vagrant box add hashicorp/precise64
vagrant box remove hashicorp/precise64

vagrant status ( to check if the vm is running)
vagrant package --output mybox.box

vagrant box add mybox mybox.box

vagrant plugin install vagrant-vbguest

```
